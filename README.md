### 사이트 선정 이유

- 대기업 홈페이지: 대기업 홈페이지라서 오류가 적을 것이라고 생각했고 구조도 잘 짜여져있을 것이라 생각했습니다.
- 반응형 웹사이트: 반응형 작업이 가능했습니다.
- 스와이퍼, 애니메이션(transition) 적용: 수업에서 배운 내용을 적용해볼 수 있을 것이라 생각했습니다.
